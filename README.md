# 🎓 Advanced Student Analyzer (NumPy)

## 📌 Description

This project analyzes student performance using NumPy arrays. It processes marks data and generates insights such as totals, averages, rankings, grades, and pass/fail status.

## 🚀 Features

* Student-wise analysis (total, average, percentage)
* Ranking system (topper to weakest)
* Grade classification (A, B, C, F)
* Pass/Fail detection with failed subjects
* Subject-wise performance analysis
* Class statistics (average, highest, lowest)

## 🛠️ Tech Stack

* Python
* NumPy

## ▶️ How to Run

```bash
python main.py
```
